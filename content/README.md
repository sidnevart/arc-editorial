# Content

Здесь лежат редакционные материалы по проекту `arc`.

Структура:

- `posts/` — короткие посты для канала, X, Telegram, LinkedIn или VC
- `articles/` — более развёрнутые статьи
- `linkedin/` — отдельная серия постов под LinkedIn с более деловой подачей
- `campaigns/` — готовые контент-пакеты под конкретные narrative arcs и каналы

Текущая основная кампания:

- `campaigns/2026-03-28-arc-desktop-vnext/`

Во всех материалах, где нужна визуализация, есть секция `Промпт для картинки`.
Промпты написаны так, чтобы их можно было отдавать в GPT отдельно, без внешнего контекста.
