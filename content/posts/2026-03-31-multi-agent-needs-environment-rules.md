# Пост: multi-agent без environment rules быстро превращается в кашу

Пока агент один, почти всё кажется нормальным.

Как только агентов становится много, внезапно выясняется, что главный вопрос не “какого сделать следующим”, а “кто вообще чем владеет”.

Hooks, memory scopes, permissions, runtime policy.

Если у этого нет границ, платформа начинает течь очень быстро.
