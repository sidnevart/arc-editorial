# Пост: sandbox profile нужен не для красоты, а для доверия

Фраза `sandboxed_exec` ничего не значит, если её нельзя развернуть в конкретику.

Нормальная система должна уметь показать:

- где hook работал
- какой env получил
- какие memory scopes были разрешены
- по какому bounded profile он вообще исполнялся

И только после этого слово “sandboxed” начинает чего-то стоить.
