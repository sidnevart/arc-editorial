# Пост: noise reduction — одна из лучших метрик для context system

Мне очень нравится смотреть не только на “сколько токенов вышло в итоге”, а на:

- сколько кандидатов было
- сколько осталось
- сколько шума выбросили

Потому что именно там видно, работает ли context system как фильтр или просто красиво переупаковывает тот же мусор.
