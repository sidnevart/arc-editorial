# Пост: детали чата должны быть правдой о runtime, а не пустым drawer’ом

У правой панели в AI desktop есть два пути:

- стать мусорной зоной “на потом”
- стать местом, где реально видно живое состояние системы

Мне нравится только второй вариант.

Если уж у тебя есть `Детали чата`, там должны быть:

- запущенные симуляции
- материалы
- файлы
- полезная runtime-правда
