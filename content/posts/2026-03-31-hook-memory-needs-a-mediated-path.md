# Пост: hook memory должна идти через управляемый path

Если hooks могут писать в память как попало, memory layer очень быстро перестаёт быть доверенным.

Поэтому mediated path вроде `arc hook memory add` — это не мелочь.

Это способ сказать:

- кто писал
- куда писал
- имел ли право
- оставить audit

Вот так и появляется взрослая memory system.
