# Пост: модель без runtime — это дорогой хаос

Сильная модель без runtime очень быстро начинает:

- жечь токены
- плавать по поведению
- терять контекст
- путать факт и гипотезу

Поэтому ARC мы строим не вместо Codex, а поверх него.

Codex даёт reasoning.
ARC даёт контекст, правила, бюджет, артефакты и управляемую среду.

Вот где начинается настоящий продукт.
