# Пост: provider footprint важнее, чем просто сумма токенов

Когда budget layer умеет показать не только “сколько стоило”, но и:

- какой model реально вызвали
- по какому policy path
- с каким routing decision
- в каком provider session

ты получаешь уже не абстрактную метрику, а нормальную трассировку стоимости.
