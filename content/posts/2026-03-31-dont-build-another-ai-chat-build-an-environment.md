# Пост: не стройте ещё один AI-чат. Стройте среду.

Если честно, AI-чата как продукта уже давно недостаточно.

Интерес начинается там, где у тебя появляется среда:

- агентные правила
- context system
- rich outputs
- локальные миниаппы
- бюджетный слой
- артефакты

Вот из этого уже может вырасти настоящая платформа.
