# arc-editorial

Editorial and media repository for ARC.

## Structure

- `content/posts/` — short-form posts
- `content/articles/` — long-form articles
- `content/linkedin/` — LinkedIn-focused posts and series
- `content/campaigns/` — multi-asset launch and rollout campaigns
- `branding/` — shared branding assets

## Notes

- Filenames are normalized for GitHub-friendly URLs and predictable search.
- Content remains authored in Russian where appropriate; only file naming and repo structure were normalized.
- This repository is staged locally from the monorepo and is ready to push once the GitHub remote exists.
