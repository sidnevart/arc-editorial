# ARC Desktop VNext Media Pack

Этот пакет собирает материалы для рассказа о новом позиционировании ARC:

- chat-first workspace вместо перегруженного task manager
- built-in agents `Study / Work / Hero` как управляемые пресеты
- встроенные материалы, диаграммы, демо и мини-симуляции
- managed local runtime без обязательного Docker
- developer-only testing mode и verification layer

Структура:

- `habr-articles.md` — идеи и каркасы больших статей
- `devlog-series.md` — дневник разработки по дням
- `linkedin-posts.md` — деловые короткие посты
- `twitter-threads.md` — треды и короткие narratives для X
- `reels-ideas.md` — идеи для коротких видео
- `bigtech-outreach.md` — углы подачи для команд из бигтеха и enterprise
- `image-prompts.md` — отдельные промпты для GPT image generation

Промпты для изображений написаны так, чтобы их можно было отдавать отдельно, без доступа к репозиторию и скрытому контексту.
